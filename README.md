# fTerm

fTerm is a terminal with english syntax and natural language processing.

# About

## Getting Started

### Installing (Mac)

First, run

```
brew tap lschumm/tap
```
and then

```
brew install fterm
```

End with an example of getting some data out of the system or using it for a little demo
